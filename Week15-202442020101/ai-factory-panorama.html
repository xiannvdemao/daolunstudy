各阶段详解

🤖 Agent 编排

三 Agent 并行分析，提升代码审查效率

  • explore: 代码调用点分析
  • librarian: 外部文档验证
  • oracle: 性能风险评估

⚙️ 代码生成

AI 辅助生成代码，人工审查后提交 PR

  • AI 根据分析结果生成代码
  • 开发者审查并调整
  • 提交 PR 触发 Gate 检查

🛡️ Harness 验证

三层门禁确保代码质量

  • BP1: 编译 + 格式 + Clippy
  • BP2: 测试 + QPS 基准 (≥10,000)
  • BP3: 风险评估 + 最佳实践

🚀 CI/CD 执行

自动化构建、测试、部署流程

  • GitHub Actions 自动触发
  • 多平台交叉编译
  • 自动部署 + 监控告警

📚 知识沉淀

经验沉淀到 GBrain 知识库

  • Episode 记录优化过程
  • 更新回归检测基线
  • Pattern 库持续积累

🔄 PDCA 闭环

持续改进的自优化循环

  • Plan: 设定优化目标
  • Do: 实现优化方案
  • Check: Gate 验证结果
  • Act: 沉淀知识 + 下一轮
